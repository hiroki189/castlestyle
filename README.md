 ## Castlestyle
日本全国の入館できる城をを見る・投稿することができるサイトです。  ポートフォリオとして作成しました。
# URL
<http://13.114.92.47/>

![](https://i.gyazo.com/2a1a23c249a4281f20703328a6427015.jpg)

## 使用技術

・Ruby 2.5.1<br>
・Ruby on Rails 5.2.3<br>
・MySQL2<br>
・SCSS<br>・jQuery<br>
・Javascript  
・AWS

## 機能一覧

・ユーザー登録<br>
・ログイン機能(devise)<br>
・投稿機能<br>
・ページネーション(kaminari)<br>
・GoogleMap 共有
